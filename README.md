# 🌲 Forest Collector

A small 2D platformer built with **Godot Engine** where the player explores a forest, reaches a gate, and completes the level.

[![Click to Play](https://img.shields.io/badge/CLICK%20ME%20TO%20PLAY-itch.io-orange?style=for-the-badge&logo=itchdotio)](https://ambariz.itch.io/forest-collector)

 **LINK :** https://ambariz.itch.io/forest-collector  
---

## Gameplay

- Move and jump through a forest-themed level  
- Avoid falling off platforms  
- Reach the gate to complete the level  
- Press **Enter** to restart after completing the level  

---

## Controls

- **A / D or Arrow Keys** → Move left/right  
- **Space** → Jump  
- **Enter** → Restart (after level complete screen)  

---

## Features

- Simple and responsive platformer movement  
- Level completion system with UI overlay  
- Restart mechanic using input  
- Basic fall detection reset system  
- Lightweight 2D game structure  

---

## Built With

- Godot Engine  
- 2D TileMap system  
- Custom sprites and assets  
- GDScript
